# Deburring
Deburring in skeleton BW image
***
本函数的应用领域是图片中**裂缝/裂纹检测** <br>

应用**传统图像处理方法**对裂纹进行检测，在对二值图进行骨架提取后骨架上会残留<br>
许多毛刺。基于此，
>本函数的功能是对图片中提取的骨架毛刺进行抑制<br>
>使得检测的裂纹更`平滑、合理`<br>
